# ADR-20260703-02: Proxy Backend para API Keys — Cloudflare Worker Design

**ID:** `ARCH-20260703-02`

---

## 🎯 Contexto

**Problema:** Las API Keys de Gemini **NUNCA** pueden estar en el cliente (bundle, localStorage, IndexedDB, variables de build `VITE_`). Cualquier exposición = compromiso de cuenta Google Cloud, facturación no autorizada, abuso de cuotas.

**Solución:** **Cloudflare Worker Proxy** — Serverless, edge, secrets seguros, rate limiting nativo, CORS resuelto, logs de auditoría.

---

## 🏗️ ARQUITECTURA DEL PROXY

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         CLOUDFLARE WORKER PROXY                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  RUTAS:                                                                    │
│  ├─ POST /api/gemini/generateContent     → Gemini generateContent        │
│  ├─ POST /api/gemini/generateVideo       → Veo generateVideo             │
│  ├─ GET  /api/gemini/operations/{name}   → Veo pollOperation             │
│  ├─ POST /api/gemini/generateImage       → Imagen 3 generateImage        │
│  ├─ POST /api/gemini/analyzeImage        → Gemini Vision analyzeImage    │
│  └─ POST /api/gemini/synthesizeSpeech    → Gemini TTS                    │
│                                                                             │
│  SECRETS (Cloudflare Dashboard):                                          │
│  ├─ GEMINI_API_KEY                   → Bearer token para todas las rutas │
│  ├─ RATE_LIMIT_RPM                   → Requests por minuto por IP (def: 10)│
│  ├─ ALLOWED_ORIGINS                  → Lista dominios permitidos (CORS)   │
│  └─ LOG_LEVEL                        → debug/info/warn/error             │
│                                                                             │
│  MIDDLEWARE CADENA:                                                       │
│  1. CORS Preflight (OPTIONS) → 204 + headers                             │
│  2. Rate Limit (IP + User-Agent) → 429 si excede                         │
│  3. Validación Content-Type: application/json                            │
│  4. Inyección Authorization: Bearer ${GEMINI_API_KEY}                    │
│  5. Forward a Gemini API (https://generativelanguage.googleapis.com)     │
│  6. Parse Response:                                                       │
│     ├─ 200 OK → Forward body + headers (sin auth)                        │
│     ├─ 400/401/403 → Log + forward error (sin key details)               │
│     ├─ 429 → Retry-After header + 429 al cliente                         │
│     ├─ 5xx → Retry interno 2x → 502 si persiste                          │
│     └─ Safety Filters → Parse safetyRatings → añadir header X-Safety    │
│  7. Logging: requestId, IP, route, latency, status, safetyFlags          │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### src/index.ts (Worker Principal)

```typescript
// src/index.ts
import { Hono } from 'hono';
import { cors } from 'hono/cors';
import { rateLimiter } from 'hono-rate-limiter';
import { logger } from 'hono/logger';
import { secureHeaders } from 'hono/secure-headers';

type Bindings = {
  GEMINI_API_KEY: string;
  RATE_LIMIT_KV: KVNamespace;
  RATE_LIMIT_RPM: string;
  ALLOWED_ORIGINS: string;
  LOG_LEVEL: string;
};

const app = new Hono<{ Bindings: Bindings }>();

// Middleware global
app.use('*', logger());
app.use('*', secureHeaders());
app.use('*', cors({
  origin: (origin, c) => {
    const allowed = c.env.ALLOWED_ORIGINS.split(',').map(s => s.trim());
    return allowed.includes(origin) ? origin : null;
  },
  allowMethods: ['POST', 'GET', 'OPTIONS'],
  allowHeaders: ['Content-Type', 'Authorization'],
  maxAge: 86400,
  credentials: false,
}));

// Rate Limiter por IP + User-Agent
app.use('/api/gemini/*', async (c, next) => {
  const ip = c.req.header('CF-Connecting-IP') || 'unknown';
  const ua = c.req.header('User-Agent') || 'unknown';
  const key = `rl:${ip}:${hash(ua).slice(0, 8)}`;
  
  const limiter = rateLimiter({
    windowMs: 60_000, // 1 minuto
    limit: Number(c.env.RATE_LIMIT_RPM),
    keyGenerator: () => key,
    store: {
      async get(key) { return (await c.env.RATE_LIMIT_KV.get(key)) ? parseInt(await c.env.RATE_LIMIT_KV.get(key)!) : 0; },
      async set(key, value) { await c.env.RATE_LIMIT_KV.put(key, value.toString(), { expirationTtl: 60 }); },
      async increment(key) { const current = await this.get(key); await this.set(key, current + 1); return current + 1; }
    },
    handler: (c) => c.json({ error: 'Rate limit exceeded', retryAfter: 60 }, 429, { 'Retry-After': '60' })
  });
  return limiter(c, next);
});

// Helper: Forward a Gemini con API Key inyectada
async function forwardToGemini(c: any, path: string, body: any) {
  const requestId = crypto.randomUUID();
  const start = performance.now();
  
  try {
    const geminiUrl = `https://generativelanguage.googleapis.com/v1beta${path}?key=${c.env.GEMINI_API_KEY}`;
    
    const response = await fetch(geminiUrl, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'User-Agent': 'Bridge-Creative-Engine/1.0',
        'X-Request-ID': requestId,
      },
      body: JSON.stringify(body),
      // Cloudflare: 30s timeout, 100MB body limit
    });
    
    const latency = performance.now() - start;
    const responseBody = await response.json();
    
    // Parse Safety Ratings
    const safetyFlags = parseSafetyRatings(responseBody);
    if (safetyFlags.length > 0) {
      c.header('X-Safety-Flags', safetyFlags.join(','));
      console.warn({ requestId, safetyFlags, path });
    }
    
    // Log estructurado
    console.info({
      requestId,
      path,
      status: response.status,
      latencyMs: Math.round(latency),
      ip: c.req.header('CF-Connecting-IP'),
      safetyFlags: safetyFlags.length > 0 ? safetyFlags : undefined,
    });
    
    // Headers de respuesta
    c.header('X-Request-ID', requestId);
    c.header('X-Latency-Ms', Math.round(latency).toString());
    
    return c.json(responseBody, response.status);
    
  } catch (error) {
    const latency = performance.now() - start;
    console.error({ requestId, path, error: error.message, latencyMs: Math.round(latency) });
    return c.json({ error: 'Proxy error', detail: error.message }, 502);
  }
}

// RUTAS

// 1. Generate Content (LLM structuring, copy, etc.)
app.post('/api/gemini/generateContent', async (c) => {
  const body = await c.req.json();
  return forwardToGemini(c, '/models/gemini-2.5-pro:generateContent', body);
});

// 2. Generate Video (Veo 3.1)
app.post('/api/gemini/generateVideo', async (c) => {
  const body = await c.req.json();
  return forwardToGemini(c, '/models/veo-3.1:generateVideo', body);
});

// 3. Poll Operation (Veo long-running)
app.get('/api/gemini/operations/:name', async (c) => {
  const name = c.req.param('name');
  const requestId = crypto.randomUUID();
  const start = performance.now();
  
  try {
    const geminiUrl = `https://generativelanguage.googleapis.com/v1beta/${name}?key=${c.env.GEMINI_API_KEY}`;
    const response = await fetch(geminiUrl, {
      headers: { 'User-Agent': 'Bridge-Creative-Engine/1.0', 'X-Request-ID': requestId }
    });
    const body = await response.json();
    
    c.header('X-Request-ID', requestId);
    c.header('X-Latency-Ms', Math.round(performance.now() - start).toString());
    return c.json(body, response.status);
  } catch (error) {
    return c.json({ error: 'Proxy error', detail: error.message }, 502);
  }
});

// 4. Generate Image (Imagen 3)
app.post('/api/gemini/generateImage', async (c) => {
  const body = await c.req.json();
  return forwardToGemini(c, '/models/imagen-3.0-generate-002:predict', body);
});

// 5. Analyze Image (Gemini Vision)
app.post('/api/gemini/analyzeImage', async (c) => {
  const body = await c.req.json();
  return forwardToGemini(c, '/models/gemini-2.5-pro-vision:generateContent', body);
});

// 6. Synthesize Speech (TTS)
app.post('/api/gemini/synthesizeSpeech', async (c) => {
  const body = await c.req.json();
  return forwardToGemini(c, '/models/gemini-2.5-flash-preview-tts:generateContent', body);
});

// Health Check
app.get('/health', (c) => c.json({ status: 'ok', service: 'bridge-gemini-proxy', timestamp: Date.now() }));

// 404
app.notFound((c) => c.json({ error: 'Not found' }, 404));

// Error Handler
app.onError((err, c) => {
  console.error({ error: err.message, stack: err.stack });
  return c.json({ error: 'Internal server error' }, 500);
});

export default app;

// Utilidades
function parseSafetyRatings(response: any): string[] {
  const flags: string[] = [];
  const candidates = response?.candidates || [];
  for (const candidate of candidates) {
    const ratings = candidate?.safetyRatings || [];
    for (const rating of ratings) {
      if (rating.probability === 'HIGH' || rating.probability === 'MEDIUM') {
        flags.push(`${rating.category}:${rating.probability}`);
      }
    }
  }
  return flags;
}

function hash(str: string): string {
  let hash = 0;
  for (let i = 0; i < str.length; i++) {
    hash = ((hash << 5) - hash) + str.charCodeAt(i);
    hash |= 0;
  }
  return Math.abs(hash).toString(16);
}
```

## 🔧 CONFIGURACIÓN EN CLIENTE (services/gemini/client.ts)

```typescript
// services/gemini/client.ts
const PROXY_BASE = import.meta.env.VITE_PROXY_BASE || '/api/gemini'; // Relativo en prod, proxy en dev

interface ProxyRequestOptions {
  path: string;
  body: any;
  signal?: AbortSignal;
}

export class GeminiProxyClient {
  private baseUrl: string;
  private defaultHeaders: HeadersInit;

  constructor(baseUrl: string = PROXY_BASE) {
    this.baseUrl = baseUrl;
    this.defaultHeaders = {
      'Content-Type': 'application/json',
      'Accept': 'application/json',
    };
  }

  async request<T>({ path, body, signal }: ProxyRequestOptions): Promise<T> {
    const controller = new AbortController();
    const timeoutId = setTimeout(() => controller.abort(), 180_000); // 3 min timeout Veo
    
    const abortSignal = signal ? AbortSignal.any([signal, controller.signal]) : controller.signal;

    try {
      const response = await fetch(`${this.baseUrl}${path}`, {
        method: 'POST',
        headers: this.defaultHeaders,
        body: JSON.stringify(body),
        signal: abortSignal,
      });

      clearTimeout(timeoutId);

      if (!response.ok) {
        const error = await response.json().catch(() => ({ error: 'Unknown error' }));
        throw new GeminiProxyError(response.status, error.error || 'Request failed', error);
      }

      return response.json();
    } catch (error) {
      clearTimeout(timeoutId);
      if (error instanceof DOMException && error.name === 'AbortError') {
        throw new GeminiProxyError(408, 'Request timeout (3min)', { timeout: true });
      }
      throw error;
    }
  }

  // Métodos de conveniencia
  generateContent(body: any) { return this.request({ path: '/generateContent', body }); }
  generateVideo(body: any) { return this.request({ path: '/generateVideo', body }); }
  pollOperation(name: string) { return this.request({ path: `/operations/${name}`, method: 'GET' }); }
  generateImage(body: any) { return this.request({ path: '/generateImage', body }); }
  analyzeImage(body: any) { return this.request({ path: '/analyzeImage', body }); }
  synthesizeSpeech(body: any) { return this.request({ path: '/synthesizeSpeech', body }); }
}

export class GeminiProxyError extends Error {
  constructor(
    public status: number,
    message: string,
    public details: any
  ) {
    super(message);
    this.name = 'GeminiProxyError';
  }
}
```

## 🛡️ SEGURIDAD: Threat Model

| Amenaza | Mitigación en Proxy |
|---------|---------------------|
| **XSS roba key localStorage** | Key NUNCA en cliente. Proxy inyecta server-side. |
| **Bundle expone VITE_KEY** | No hay VITE_GEMINI_KEY. Cliente usa `/api/gemini/*` relativo. |
| **Abuso de cuota (botnets)** | Rate limit 10 RPM/IP en KV. Cloudflare WAF opcional. |
| **Replay attacks** | Request-ID único, logs de auditoría, timestamps. |
| **Data exfiltration** | Proxy NO logea bodies completos, solo metadata + safety flags. |
| **CORS bypass** | `ALLOWED_ORIGINS` estricta. Credentials: false. |
| **DoS en Worker** | Cloudflare: 30s CPU limit, 100MB body, rate limit en edge. |

- `latencyMs > 30000` (Veo timeout risk)
- `status >= 500` rate > 1%/5min

- `rateLimitHits` > 10/min
